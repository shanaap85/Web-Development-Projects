# 🔒 Update Password

A responsive password update form built using **HTML5**, **CSS3**, and **JavaScript**. The application validates user input by checking required fields and ensuring that the new password and confirmation password match before allowing the password to be updated.

## ✨ Features

- Responsive user interface
- Password update form
- Required field validation
- Password confirmation validation
- Dynamic validation messages
- Clean and user-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

1. Clone or download this repository.
2. Open `index.html` in your preferred web browser.
3. Enter the required details.
4. Try different password combinations to observe the validation behavior.
